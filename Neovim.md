This is a text editor that can be extremely powerful thanks to Lua scripting!

## Installing neovim:
###### To install on Arch Linux:
```sudo pacman -S neovim```
###### To install on Ubuntu/Debian:
```sudo apt install neovim```
###### To install on Fedora:
```rpm install neovim```
###### To Install on Mac:
```brew install neovim```
